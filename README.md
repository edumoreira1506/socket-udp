## socket-udp

This project is a academic task from UTFPR.

## Dependencies

```
NodeJS >= 10.17
Yarn >= 1.21
```

## Setup

Clone the project.
```console
$ git clone git@github.com:edumoreira1506/socket-udp.git
$ cd socket-udp
```


## Available Scripts

In the project directory, you can run:

### `yarn start:server`

Turn on the socket UDP server.

### `yarn start:client`

Turn on the socket UDP client.
